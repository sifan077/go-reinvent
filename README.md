# Go 从零造轮子 · 学习笔记

> 尽量用标准库，少依赖，自己动手写核心逻辑

通过动手实现一些常见的基础组件，加深对 Go 语言特性和底层原理的理解。

## 学习路线

### 一、入门阶段 — 熟悉语法和封装

| # | 模块 | 核心知识点 |
|---|------|-----------|
| 1 | 彩色分级日志库 | 日志级别、时间格式化、ANSI 彩色输出、自定义格式 |
| 2 | 日志轮转库 | 按大小切割、按日期切割、旧日志自动清理 |
| 3 | 通用工具库 | 时间工具、字符串工具、泛型集合（过滤/去重/分组/交集并集） |

### 二、进阶阶段 — 练习并发和数据结构

| # | 模块 | 核心知识点 |
|---|------|-----------|
| 4 | LRU 本地缓存 | 哈希表 + 双向链表、过期淘汰、定时清理 |
| 5 | 单机限流器 | 令牌桶、漏桶、滑动窗口 |
| 6 | 协程池 | 任务队列、并发控制、任务超时、panic 捕获 |

### 三、中级组件 — 有一定实用性

| # | 模块 | 核心知识点 |
|---|------|-----------|
| 7 | 通用连接池 | 空闲复用、最大连接数、超时机制、健康检查 |
| 8 | 简易 Web 框架 | 基于 net/http 自研路由、中间件链、路由分组 |
| 9 | 分布式基础工具 | 雪花算法 ID、一致性哈希、简易分布式锁 |

### 四、高阶组件 — 挑战一下复杂度

| # | 模块 | 核心知识点 |
|---|------|-----------|
| 10 | 极简 ORM | 反射映射 SQL、链式调用、基础 CRUD |
| 11 | TCP 网络框架 | 粘包拆包、消息协议封装、连接管理 |
| 12 | 极简 RPC 框架 | 基于 TCP 自研协议、序列化、负载均衡 |

## 推荐学习顺序

```
日志库 → 日志轮转 → 泛型工具库 → LRU缓存 → 限流器 →
协程池 → 连接池 → Web框架 → 雪花ID/一致性哈希 →
ORM → TCP框架 → RPC
```

## 项目结构

```
go-reinvent/
├── cmd/            # 各模块的示例入口
├── internal/       # 内部实现
├── pkg/            # 可复用的轮子包
├── scripts/        # 辅助脚本
└── plan/           # 学习计划与笔记
```

## 原则

- 尽量不依赖第三方库，核心逻辑自己写
- 每个模块尽量独立，附带简单的示例代码
- 关键地方加注释，记录一下思路和踩过的坑

## 运行环境

- Go 1.21+
